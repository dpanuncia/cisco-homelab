# Homelab Roadmap — CCNA & Enterprise Networking
**Goal:** CCNA on December 6, 2026 | West Michigan IT/Cybersecurity Career
**Hardware:** Dream Machine Pro · 2x Cisco 2900 · 2x Cisco 2960-S PoE+

---

## Phase 1 — Foundation (June–July)
Get devices talking. Build the habit of documenting everything.

| Lab | Topic | CCNA Domain | Enterprise Angle |
|-----|-------|-------------|-----------------|
| 01 | Network Topology & IP Planning | Fundamentals | Done |
| 02 | Console Access & Basic Navigation | Fundamentals | How admins first touch a device |
| 03 | Device Hardening (SSH, passwords, banners) | Security | Zero-trust baseline every corp uses |
| 04 | VLANs & Trunking | Network Access | How Corewell Health isolates patient data |
| 05 | Spanning Tree Protocol (STP) | Network Access | Prevent loops in multi-switch networks |

## Phase 2 — Routing & Connectivity (August–September)
Get traffic flowing across routers. This is the heart of CCNA.

| Lab | Topic | CCNA Domain | Enterprise Angle |
|-----|-------|-------------|-----------------|
| 06 | Inter-VLAN Routing (Router-on-a-Stick) | IP Connectivity | Steelcase floor-to-floor traffic |
| 07 | Static Routing | IP Connectivity | Small office/branch links |
| 08 | OSPF Dynamic Routing | IP Connectivity | How large campuses route internally |
| 09 | DHCP (Cisco router as server) | IP Services | Every device on every network |
| 10 | NAT/PAT (Internet access) | IP Services | How your LAN hits the internet |

## Phase 3 — Security & Services (October–November)
Lock things down. This is what gets you hired in cybersecurity roles.

| Lab | Topic | CCNA Domain | Enterprise Angle |
|-----|-------|-------------|-----------------|
| 11 | Access Control Lists (ACLs) | Security | Gordon Food Services network segmentation |
| 12 | EtherChannel (LACP) | Network Access | Redundant uplinks in server rooms |
| 13 | Port Security | Security | Preventing rogue devices on Amway's network |
| 14 | Syslog & SNMP Monitoring | IP Services | How NOC teams see network health |
| 15 | Capstone — Full Enterprise Simulation | All | Everything wired together, documented |

---

## Study Tips
- Do each lab on real hardware, then simulate it in Cisco Packet Tracer too
- Write the writeup BEFORE you configure — forces you to think first
- Post every lab to your website — your portfolio for interviews
- Screenshot every error and explain it — shows growth to employers
